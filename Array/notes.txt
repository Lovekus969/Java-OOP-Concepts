Declaring arrays
● Creating arrays
● Arrays of primitive and reference type
● Initialization of elements
● Multidimensional arrays


Declaring Arrays :=> 
      Declaring an array means telling Java what type of elements it will store.
                                                      Syntax:
                                                      dataType[] arrayName;
Creating Arrays:=>
      After declaration, we create the array in memory using new.

                                                      Syntax:
                                                      arrayName = new dataType[size];

 Arrays of Primitive and Reference Types :=>
  Primitive Type Array: stores actual values.
                  int[] speeds = {280, 305, 320};
                  System.out.println(speeds[1]); // 305


Reference Type Array: stores references (memory addresses) to objects.

            BMW[] myCars = new BMW[2];
            myCars[0] = new M4CS(280, 4.1);
            myCars[1] = new M5CS(305, 3.8);

Initialization of Elements:=>
      Primitive arrays: default values: int = 0, boolean = false, etc.

            int[] arr = new int[3];  // elements = [0,0,0]
            arr[0] = 10;
            arr[1] = 20;
            arr[2] = 30;

      Reference arrays: default values = null

            BMW[] cars = new BMW[2];  // cars[0] and cars[1] are null initially
            cars[0] = new M4CS(280, 4.1);
            cars[1] = new M5CS(305, 3.8);

Multidimensional Arrays
Definition: Array of arrays. Think of it like a table with rows and columns.
